# Consegna S1/L1
## 1. Installazione della Macchina Vittima Metasploitable
- Lo studente dovrà installare la macchina vittima Metasploitable. Può essere scaricata da questo link. https://sourceforge.net/projects/metasploitable/
-  Dopo aver scaricato il file, installarlo su VirtualBox seguendo le istruzioni fornite.
## 2.  Bonus: Installazione di Windows 10 
   - Per un'esperienza aggiuntiva, gli studenti possono seguire la stessa procedura per installare Windows 10 su VirtualBox.
